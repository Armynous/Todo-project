# Todo App

This is a simple Todo application built with React, DaisyUI, and Material-UI.

## Features

- View a list of todos
- Add a new todo
- Update an existing todo
- Delete a todo

## Technologies Used

- React
- DaisyUI
- Material-UI
- Axios
